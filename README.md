# 📡 RSS Converter
### *YouTube → RSS/JSON. Технологично, удобно, почти законно.*

[![Version](https://img.shields.io/badge/version-1.0-blue.svg)]()
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Знакомая ситуация: у тебя есть крутой YouTube-канал, а твой сайт на российском хостинге его просто не видит. Не потому, что ты что-то нарушаешь, а потому что IP-адрес попал под «раздачу». Обидно, когда твой же контент не может попасть на твой же сайт.

**Этот скрипт решает проблему.** Он достанет RSS (или JSON) с твоего канала или плейлиста, даже если хостинг и YouTube живут в разных вселенных. Всё, что нужно — небольшой технический ритуал с Cloudflare Worker и секретным токеном. Никакой магии, только правильные прокси.

**Осторожно:** это не «прокси для всех». Это персональный мост для твоих данных. Без токена никто не пройдёт.

---

## 🚀 Быстрый старт (для нетерпеливых)

1. **Создайте Cloudflare Worker.** Назовите его, например, `ru-proxy`.
2. **Добавьте секретную переменную** `PROXY_TOKEN` с любым сложным значением.
3. **Скопируйте код** из `worker.js` в вашего Worker'а и запустите его.
4. **Скопируйте папку `plugins/tcse/rss-converter/`** к себе на сайт.
5. **Пропишите в `config.php`** адрес вашего Worker'а и ваш секретный токен.
6. **Используйте!** Просто откройте в браузере ссылку для канала или плейлиста.

> ✨ **Живой пример:**
> `https://ваш-сайт.ru/plugins/tcse/rss-converter/youtube.php?action=channel&id=UC******&format=rss`

[Подробная инструкция с картинками и кнопками «копировать» находится здесь.]([[https://github.com/tcse/rss-converter/blob/main/plugins/tcse/rss-converter/index.html](https://online.tcse-cms.com/demo/rss-converter/index.html)](https://online.tcse-cms.com/demo/rss-converter/index.html))

## 🛠️ Как это работает (очень просто)

Ваш сайт на российском хостинге -> обращается к PHP-скрипту -> скрипт идёт к вашему Cloudflare Worker -> Worker идёт на YouTube -> YouTube отдаёт данные -> Worker отдаёт их скрипту -> скрипт отдаёт вам. Ничего лишнего. Всё по делу.

## ⚠️ Важные нюансы

*   **Токен — всему голова.** Worker без токена не работает. Не потеряйте его.
*   **Кеширование.** Данные кешируются на час, чтобы не дергать YouTube по пустякам.
*   **Лимиты.** На бесплатном тарифе Cloudflare — 100 000 запросов в день. Обычному сайту за глаза.
*   **Закон.** Инструмент предназначен для доступа к вашему собственному контенту. Не надо использовать его для чужих каналов или для обхода блокировок в других целях.
